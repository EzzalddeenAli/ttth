# ttth
## services
While ttth supports a growing list of pre-configured services, you can as well add custom urls yourself (using the CUSTOM URL type).
The following services are currently supported:

* Discord
* Dropbox
* Freenode
* GitHub
* Gitter
* Google Calendar
* Google Contacts
* Google Duo
* Google Drive
* Google Keep
* Google Mail
* Google Messages
* Google Photos
* ICQ
* LinkedIn
* Mattermost
* Messenger
* Microsoft Outlook
* Microsoft Teams
* Nextcloud
* Reddit Chat
* Riot.im
* Skype
* Slack
* Steam Chat
* Telegram
* Threema
* Twitter
* WeChat
* Wire
* WhatsApp
* Xing
